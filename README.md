# Spanish Culture Quiz

# Description
- This quiz test users knowledge of Spanish culture and history. The quiz presents multiple choice questions and gives feedback based on if it is correct or incorrect, along with a final score.

# Features
- Multiple choice based questions
- Feedback
- Score
- Play Agian and Next button
